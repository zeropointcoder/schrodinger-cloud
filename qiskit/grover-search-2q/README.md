# Grover’s Algorithm – Search (2 Qubits)

Implement `Grover’s Search` algorithm using **Qiskit**, for 2-qubit systems.

It marks a specific target state (e.g. `|11⟩`) and amplifies its probability using Grover’s quantum amplitude amplification.

## Steps
- oracle
- diffuser
- grover
 
## Requirements
```bash
pip3 install -r requirements.txt
```

## Run
```bash
python3 grover_2qubit.py
```
