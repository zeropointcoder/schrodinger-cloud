# Grover’s Algorithm – Search (3 Qubits)

Implement `Grover’s Search` algorithm using **Qiskit**, for 3-qubit systems.

It marks a specific target state (e.g. `|101⟩`) and amplifies its probability using Grover’s quantum amplitude amplification.

## Steps
- oracle
- diffuser
- grover
 
## Requirements
```bash
pip3 install -r requirements.txt
```

## Run
```bash
python3 grover_3qubit.py
```
